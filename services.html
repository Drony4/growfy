    Annotation Анотація Аннотация

    The main page has been developed using a responsive and adaptive layout based on a provided educational design template. The following technologies have been utilized: HTML5, CSS3, JS, SASS, Flexbox, and BEM methodology principles.

    The main page includes a navigation menu, which transforms into a burger menu on mobile devices. When scrolling the page, the current section is highlighted based on its visibility. Additionally, certain elements, such as buttons, are rearranged for optimal display on various screen sizes.

    The page showcases the ability to animate elements upon their appearance in the user's view. This adds dynamism and visual appeal to the content.

    The "Testimonials" section features a simple slider that can be controlled by hovering on devices with a cursor and by dragging on devices without one.

    In the footer, elements are adapted to ensure a user-friendly display on different devices. Some elements are transformed into collapsible spoilers, reducing the amount of displayed information and providing a compact footer layout.

    The form includes basic field validation to ensure correct input before submission. Please note that form submission is not functional as there is no backend component to handle the data.

    This overview highlights the usage of modern technologies and functionality that make the web page interactive, responsive, and visually engaging for users.

    Головна сторінка була розроблена за допомогою навчального макету і має адаптивний дизайн. Для її створення були використані наступні технології: HTML5, CSS3, JS, SASS та Flexbox, а також принципи методології БЕМ.

    На головній сторінці є навігаційне меню, яке на мобільних пристроях перетворюється на бургер-меню. Під час прокрутки сторінки підсвічується поточний розділ, який знаходиться в полі зору користувача. Під час адаптації контенту деякі елементи, наприклад, кнопки, переміщуються для оптимального відображення на різних екранах.

    Демонструється можливість анімації елементів при їх з'явленні в полі зору користувача. Це додає динамічності та привабливості до контенту сторінки.

    У розділі "Testimonials" використовується простий слайдер, який управляється наведенням миші на пристроях із покажчиком та перетягуванням на пристроях без нього.

    У футері елементи адаптуються для зручного відображення на різних пристроях. Деякі з них перетворюються в спойлери, що допомагає скоротити обсяг відображуваної інформації та забезпечити компактність футера.

    У формі перед надсиланням на сервер виконується проста валідація полів для перевірки коректності їх заповнення. Зверніть увагу, що надсилання форми не відбувається, оскільки відсутній компонент бекенду, який обробляє дані.

    Главная страница была разработана с использованием учебного макета и имеет отзывчивую адаптивную верстку. В ее создании были использованы следующие технологии: HTML5, CSS3, JS, SASS и Flexbox, а также принципы методологии БЭМ.

    На главной странице представлено меню навигации, которое на мобильных устройствах преобразуется в меню-бургер. При прокрутке страницы подсвечивается текущая секция, которая находится в поле зрения пользователя. В процессе адаптации контента некоторые элементы, например, кнопки, переносятся для оптимального отображения на разных экранах.

    Демонстрируется возможность анимации элементов при их появлении в поле зрения пользователя. Это добавляет динамичности и привлекательности к контенту страницы.

    В разделе "Testimonials" использован простой слайдер, который управляется наведением мыши на устройствах с наличием указателя и перетаскиванием на устройствах без него.

    В футере элементы адаптируются для удобного отображения на разных устройствах. Некоторые из них превращаются в спойлеры, что помогает сократить объем отображаемой информации и обеспечить компактность футера.

    В форме, перед отправкой на сервер, выполняется простая валидация полей для проверки корректности заполнения. Учтите, что отправка формы не происходит, так как отсутствует бэкенд-компонент, который обрабатывает данные.
